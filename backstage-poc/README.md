# [Backstage](https://backstage.io)

This is your newly scaffolded Backstage App, Good Luck!

To start the app, run:

```sh
yarn install
yarn start
```

## Docker

To build and run with Docker, see **[DOCKER.md](./DOCKER.md)**. From the repo root:

```sh
yarn docker:build   # build image (host build + docker)
yarn docker:run    # run container at http://localhost:7007
```
